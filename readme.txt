#### Pre-Github Actions-driven workflow

Running locally with hot reloading:
- npm install
- npm start
Will start a dev server at a localhost address.

# Save changes to source code:
git add .
git commit -m "message"
git push

To publish updates to the live site at cam-powell.com:
npm run deploy
